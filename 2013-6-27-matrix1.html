Matrix 矩阵算法之一

Published by: 2013-06-06

使用原生Python编写表格，没有一个可用的类库， 有的类库，适用于科学计算，还有的就是wxPython， 好像没有编译到移动平台上使用，比如IOS或Android。 决定边学边写，写一个python的表格。

首先，来看几个示例图：

图1:

计算矩阵点，得出这个公式

Gm,n = Pmn

我们来使用程序来算一下

def martix(r, c):
    x = [i % c for i in range(r*c)]
    y = [i / c for i in range(r*c)]
    z = {}
    z['x'] = x
    z['y'] = y
    return z

这段代码输出：

>>> martix(1,3)
{'y': [0, 0, 0], 'x': [0, 1, 2]}
>>> martix(2,3)
{'y': [0, 0, 0, 1, 1, 1], 'x': [0, 1, 2, 0, 1, 2]}
>>> martix(3,3)
{'y': [0, 0, 0, 1, 1, 1, 2, 2, 2], 'x': [0, 1, 2, 0, 1, 2, 0, 1, 2]}

如果将x与y对应，正好是需要的数据，一一对应上。

要计算这个矩阵有多少个单元格呢？从在看上图1-3中可以寻找出规律。 我就算出来它的公式，如下：

Gm,n = m-1 * n-1

下面我再写一个程序，计算单元格的数量：

def sum_cells(r,c):
    if r or c > 0:
        return (r-1) * (c-1)

继续使用上面的示例输出：

>>> sum_cells(1,3)
0
>>> sum_cells(2,3)
2
>>> sum_cells(3,3)
4
