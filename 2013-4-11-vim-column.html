vim 80个字符提示线

Thu 11 April 2013
By timothy

In vim.

tags: vim

在看到这篇文章， 觉得还是需要规范来的比较好， 平时使用的编辑器是vim，配置vim的80个字符提示线，对规范就很有帮助了；

" VIM 7.3+ has support for highlighting a specified column.
if exists('+colorcolumn')
    set colorcolumn=80
else
    " Emulate
    au BufWinEnter * let w:m2=matchadd('ErrorMsg', '\%80v.\+', -1)
endif

这样的提示线不扰民，nice；
