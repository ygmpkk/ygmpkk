Javascript模块化开发

Thu 07 March 2013
By timothy

In Programming.

tags: javascript

Translations: en

在这之前，对Javascript的认识并不多，也没有深入的理解Javascript编程语言，觉得Javascript就是干一下简单的事情，自从看了一本书《Javascript语言精粹》，很薄的一本书，却能够将Javascript的美表达的淋漓精致，对Javascript有很深刻的认识；但除了Javascript和PHP有语言精粹类的书籍，也希望其他的语言也能够有这样优秀的数据。

模块化的编程，能够将程序很好的解耦，这样的方法能够将程序员从变幻莫测的需求变更中解救出来； Javascript的语法非常的松散，从其他语言转过来的只要看看语言精粹就可以快速的掌握它，下面的代码就是掩饰如何模块化开发；

Javascript声明一个对象非常的简单，只要：

var TIMOTHY = {
    params: false
}

对象的创建如此简单，就可以根据这样的对象，来模块化开发

(function(obj){
    TIMOTHY.language = {
        chinese: function(){
            alert('this is Chinese');
        },

        english: function(){
            alert('this is English')
        },

        china: function(){
            alert(TIMOTHY.language.chinese);
        },
    }
})();

这里仅仅演示了简单的代码，更严谨的代码参考CommonJs的，对Javascript的代码编写很大的帮助
