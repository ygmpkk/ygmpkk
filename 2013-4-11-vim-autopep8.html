Vim编辑时执行autopep8

Thu 11 April 2013
By timothy

In vim.

tags: vimpython

什么是PEP8：这是专门针对python的编程规范，还有代码风格； 很不错的一个工具，

首先需要安装pep8和autopep8

easy_install方式安装

# easy_install -U pep8 autopep8

pip方式安装

# pip install pep8 autopep8

在.vimrc文件加入

map \pep8 :call FormartSrc()<CR>
"定义FormartSrc()                                                               
func FormartSrc()                                                               
    exec "w"                                                                    
    if &filetype == 'c'                                                         
        exec "!astyle --style=ansi --one-line=keep-statements -a --suffix=none %"
    elseif &filetype == 'cpp' || &filetype == 'hpp'                             
        exec "r !astyle --style=ansi --one-line=keep-statements -a --suffix=none %> /dev/null 2>&1"
    elseif &filetype == 'perl'                                                  
        exec "!astyle --style=gnu --suffix=none %"                              
    elseif &filetype == 'py'||&filetype == 'python'                             
        exec "r !autopep8 -i --aggressive %"                                    
    elseif &filetype == 'java'                                                  
        exec "!astyle --style=java --suffix=none %"                             
    elseif &filetype == 'jsp'                                                   
        exec "!astyle --style=gnu --suffix=none %"                              
    elseif &filetype == 'xml'                                                   
        exec "!astyle --style=gnu --suffix=none %"                              
    endif                                                                       
    exec "e! %"                                                                 
endfunc

vim的\是\<Leader> map \pep8 可以替换成你的想要映射的按键比如F11， 我这样的映射是因为MAC OS X的Fxx的都被占用了，format又不是每天都用， 需要按\键，再输入pep8就可以执行命令而来
